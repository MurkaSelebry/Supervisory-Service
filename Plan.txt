План

I. База данных
	1. Таблица обьектов: 
	новая таблица под каждый тип обьекта
	2. Таблица пользователей:
		логин, пароль (шифрация), роль/права
	3. Таблица решений:
		заголовок, описание, дата, ответственный, статус (принятое/запланированное)
II(III). Интерфейс
	1. Основная форма: отображение таблиц и дашборда;
	2. Карточка обьекта
	3. Форма для добавления/редактирования
III(II). Класс для работы с таблицами
	1. Добавление
	2. Редактирование
	3. Удаление
	1. Сортировка таблицы
	2. Фильтрация/поиск
IV. "Продвинутый функционал"
	1. Дашборд
	2. Добавление по API
	3. Добавление XML
